# Real-Time Trading Dashboard (FREE)

A fully free, real-time-ish trading dashboard that displays:

- Real-time stock price
- Reddit sentiment
- Live RSI
- Color-coded buy/sell/watch signal

## Features
- Full Python backend (Flask)
- Yahoo Finance (free)
- Reddit sentiment (free)
- Test mode for replaying historical market data
- Simple HTML dashboard
